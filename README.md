# Employee Leave & Payroll Management System

A C++ console-based application designed to simulate an enterprise HR system.
The project automates employee record management, leave tracking, and payroll
calculation using object-oriented programming principles.

## Features
- Role-based employee types (Full-Time, Intern)
- Leave application and tracking
- Salary calculation with leave-based deductions
- Menu-driven interface
- Modular OOP design

## Technologies
- C++
- Object-Oriented Programming
- STL (vector)
- File Handling (extensible)
